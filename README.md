# Domain-Adaptive-CancerEmoBERT-GraphFramework

An interpretable, domain-adaptive CancerEmoBERT and multi-graph neural framework for structured emotion Detection in cancer-specific patient narratives.
<img width="1191" height="871" alt="models" src="https://github.com/user-attachments/assets/eb94e78a-21ce-4785-89ba-9b942a5a0b54" />

## Key Components

We improve emotion detection through:

- Progressive domain adaptation: **BERT → BioBERT → CancerEmoBERT**
- Graph-based relational modeling
- Retrieval-enhanced classification
- Explainable predictions using **GNNExplainer**

## Dataset

We use the officially released **Canceremo** dataset with the original train–validation–test split to ensure fair comparison and reproducibility.
